## 🚀 How to Run the Project

### Step 1 - Extract the ZIP file

Extract the folder to your Desktop or any location.

### Step 2 - Open Terminal / Command Prompt

- Windows: Press `Win + R`, type `cmd`, press Enter
- Navigate to project folder:
  cd Desktop/ecommerce-store

### Step 3 - Install Dependencies

npm install
Wait for it to finish (it downloads all required packages).

### Step 4 - Start MongoDB

- Open MongoDB Compass and click Connect
- OR make sure MongoDB service is running on your PC

### Step 5 - Start the Server

node server.js

### Step 6 - Open the Website

Open your browser and go to:
http://localhost:5000
